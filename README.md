# virat_navigation

Package for SLAM and navigation of Clueless Virat

Usage
-----

Launch a world

```bash
roslaunch virat_gazebo virat_bounded_world.launch
```

Launch Node

```bash
roslaunch virat_navigation virat_navigation.launch map_file:=path_to_map_file
```
